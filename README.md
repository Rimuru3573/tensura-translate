Асинхронный парсер и автоматический переводчик ранобэ **«О моём перерождении в слизь» (That Time I Got Reincarnated as a Slime)**. Скрипт собирает текст и оригинальные иллюстрации с сайта, переводит контент на русский язык с помощью Google Translate и упаковывает всё в красивую, готовую для чтения электронную книгу в формате PDF с настроенными метаданными.

---

## ✨ Особенности

- 🚀 **Асинхронный движок:** На базе `nodriver` (улучшенный аналог Playwright/Selenium), который легко обходит базовые защиты сайтов и работает в разы быстрее.
- 🌐 **Автоматический перевод:** Интеграция с Google Translate для быстрого и понятного перевода новых глав.
- 🖼️ **Импорт иллюстраций:** Скрипт находит оригинальные изображения, скачивает их в обход блокировок потоков и ровно встраивает в структуру книги.
- 📄 **Готовый PDF:** На выходе получается сверстанный PDF-файл с метаданными (Автор, Название, Тема), который приятно читать на телефонах, планшетах или читалках.

## 🛠️ Инструкция по установке и запуску

### 1. Подготовка окружения
Скрипт тестировался на современных дистрибутивах Linux (включая NixOS) и Python версии 3.11+.

Для начала склонируйте репозиторий, создайте и активируйте виртуальное окружение:


```text
README.md created successfully

```bash
# Клонируем проект (или просто перейдите в папку с кодом)
cd tensura-translate

# Создаем изолированное виртуальное окружение
python -m venv .venv

# Активируем его
# Для Linux / macOS:
source .venv/bin/activate

# Для Windows (CMD):
# .venv\\Scripts\\activate.bat

```

### 2. Установка зависимостей

Установите все необходимые библиотеки одной командой через `requirements.txt`:

```bash
pip install -r requirements.txt

```

> **Примечание:** Убедитесь, что в системе установлен браузер Google Chrome или Chromium, так как `nodriver` использует его для рендеринга страниц.

### 3. Настройка тома для перевода

Перед запуском откройте файл `main.py` в любом текстовом редакторе и измените переменную `tom` на номер того тома, который вы хотите скачать и перевести:

```python
# main.py

tom = "12"  # Укажите нужный номер тома (например, "12", "13", "14")

```

### 4. Запуск парсера

Запустите скрипт через терминал:

```bash
python main.py

```

После завершения работы скрипта в корневой папке появится готовый файл книги `book.pdf`, а также файл `test.txt` с сырым текстовым переводом для проверки.
